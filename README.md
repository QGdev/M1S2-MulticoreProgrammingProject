# Projet de Programmation multi-coeurs

## Description :

Ce projet rentre dans le cadre du cours de Programmation multi-coeurs (X2I3010).
Nous avons réalisé plusieurs algorithmes, dont une implémentation de TL2, l'implémentation des threads pools réalisés durant le distanciel et le code du TP3
utilisant nos nouvelles implémentations.
